# Calculation-Methods

Все файлы с _new - дополнительное задание.
Метод Хорд рассмотрен на лекции для одномерного случая - я его переделал для n-мерного, теперь мы проводим аналогичые дейтвия, но только последовательно для каждого направления.

В двух последних лабораторных работах представлены две реализации: вычисление градиента и матрицы Гессе символьно (_sym) и численно (_num).
